Setup Guide
⚙ .env Configuration
# Twilio
TWILIO_ACCOUNT_SID=ACa9a85...
TWILIO_AUTH_TOKEN=⚠ ROTATE THIS
TWILIO_PHONE_NUMBER=+1XXXXXXXXXX

# Claude AI
ANTHROPIC_API_KEY=sk-ant-XXXXXX

# Server
BASE_URL=https://xxx.ngrok.io
COLLEGE_NAME=Your College
🔧 Twilio Webhook URLs
Incoming Call → HTTP POST
https://xxx.ngrok.io/voice
Status Callback → HTTP POST
https://xxx.ngrok.io/call-status
Set these in Twilio Console → Phone Numbers → Your Number → Voice
🚀 Quick Start
1
Install dependencies
cd ai-call-agent && npm install
2
Set up environment file
cp .env.example .env # Fill in your keys
3
Start the backend server
npm run dev
4
Expose with ngrok (public URL)
npx ngrok http 3001
5
Paste ngrok URL into Twilio Console
https://YOUR-NGROK.ngrok.io/voice
6
Call your Twilio number — AI answers!
Get Anthropic key: console.anthropic.com
